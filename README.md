# Vanilla JS Expense Tracker 💰

A lightweight, responsive web application for tracking daily expenses. Built entirely with vanilla JavaScript, HTML5, and custom CSS, this project demonstrates core frontend fundamentals including DOM manipulation, event listening, input validation, and dynamic UI updates.

## 🚀 Features

* **Dynamic DOM Manipulation:** Add and delete expenses with UI updates happening in real-time without page reloads.
* **Smart Validation:** Prevents empty submissions, negative values, and enforces a maximum individual expense limit ($10,000).
* **Real-time Calculations:** Automatically calculates and updates the total expense amount as items are added or removed.
* **Responsive Design:** Custom CSS media queries combined with Bootstrap layout classes ensure the app looks great on desktop, tablet, and mobile screens.
* **Smooth UI:** Features custom CSS animations (fade-in effects) for newly added table rows.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and form setup.
* **CSS3:** Custom styling, Flexbox layouts, and keyframe animations.
* **JavaScript (Vanilla):** Core logic, array management, and DOM interactions.
* **Bootstrap 5 (CDN):** Baseline styling and responsive grid utility.
